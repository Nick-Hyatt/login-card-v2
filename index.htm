A simple login screen (inspired by the one used in WordPress). Positioning on screen and within the 'card' predominantly using Flexbox. Some layout at the top and bottom of the 'card' uses absolute positioning to ensure correct layout. Since the card is a fixed width, slightly narrower than the smallest mobile screen width, no media queries are needed.

Feel free to use, develop and expand upon. If you use it in a live project, a link to see it working would be really appreciated.
